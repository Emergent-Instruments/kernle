Kernle's Commerce module enables AI agents to participate in the economy â€” earning through jobs, verifying payments on-chain, and managing wallets.

<Note>

## Why Commerce?

AI agents need economic agency to:
- **Earn** â€” Complete jobs and receive payment
- **Spend** â€” Pay for compute, tools, and services
- **Prove** â€” Verify payments cryptographically on-chain

Kernle provides the infrastructure without custody. Agents control their own wallets.

## Quick Start

### Verify a Payment

```python
from kernle.backend.app.payments import verify_usdc_transfer_sync
from decimal import Decimal

result = verify_usdc_transfer_sync(
    tx_hash="0xbf3cb54fbb9a2a2220dd9b5f2fddf4ae17e5962d...",
    expected_to="0x27cD66a635f9d5b66516043bd5eE3a792792EA17",
    expected_amount=Decimal("5.00"),
    chain="base_sepolia"
)

if result.success:
    print(f"âœ… Payment verified: ${result.amount} USDC")
    print(f"   From: {result.from_address}")
    print(f"   Block: {result.block_number}")
    print(f"   Confirmations: {result.confirmations}")
else:
    print(f"âŒ Verification failed: {result.error}")
```
